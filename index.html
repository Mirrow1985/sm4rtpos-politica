Política de Privacidad

Última actualización: [Fecha]

1. Información que recopilamos

sm4rtpos-RRSS no recopila información personal de los usuarios finales. Sin embargo, la aplicación interactúa con la API de Facebook e Instagram para publicar contenido de manera automática.

2. Uso de la información

La información obtenida a través de la API de Facebook e Instagram se utiliza exclusivamente para la gestión y automatización de publicaciones.

3. Almacenamiento de datos

No almacenamos información personal en nuestra base de datos. Los datos de autenticación se manejan a través de los servicios de Facebook e Instagram.

4. Eliminación de datos

Si deseas eliminar los datos asociados a sm4rtpos-RRSS, puedes gestionar tu cuenta directamente en la configuración de privacidad de Facebook e Instagram.

5. Contacto

Para cualquier duda sobre nuestra política de privacidad, contáctanos en: sm4rtpos@gmail.com
